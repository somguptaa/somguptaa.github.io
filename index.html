Food Order Management System

Full-stack application with a Spring Boot REST API backend and a dark-theme SPA frontend served via Spring Boot static resources — no external JS framework. Implements a strict 5-stage order lifecycle with domain-level enforcement.

  • 6+ RESTful APIs: full CRUD + PATCH status endpoint enforcing 5-stage lifecycle: PENDING → CONFIRMED → PREPARING → DELIVERED / CANCELLED

  • All endpoints documented with SpringDoc OpenAPI + Swagger UI for interactive testing and clear API contracts
